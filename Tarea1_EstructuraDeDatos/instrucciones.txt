Desde el método main se realiza la creación de los nodos que formarán parte de la estructura, 
se utiliza el constructor de los objetos tipo Nodo que recibe como parámetros el valor del nodo 
y el número máximo de enlaces a otros nodos que éste puede tener.
La sintaxis para la creación de nodos es:
  Nodo nombre_nodo = new Nodo("valor_nodo", numero_max_enlaces);

Donde:
  Nodo hace referencia al tipo de dato
  nombre_nodo es un identificador para el nodo
  new Nodo() indica que se construye un nuevo objeto de tipo Nodo
  "valor_nodo" recibe un valor de tipo String que tomará el nuevo Nodo
  numero_max_anlaces recibe un valor de tipo int y representa la mayor cantidad de nodos enlazados 
    que puede tener el nodo que se está creando

A partir de los nodos construidos ahora se formarán los enlaces entre ellos, con ayuda de la 
función enlazar, cuya sintaxis es:

  nombre_nodo_origen.enlazar(nombre_nodo_destino);

Donde:
  nombre_nodo_origen hace referencia al nodo desde el cual se origina la conexión
  nombre_nodo_destino se refiere al nodo que se desea conectar el nodo origen, es el último nodo
  que abarca la conexión a crear
